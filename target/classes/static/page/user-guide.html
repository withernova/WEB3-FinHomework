系统使用说明

救援管理模块

救援申请

如何填写救援申请

用户需要填写走失老人的详细信息，包括姓名、性别、走失时间和地点等内容，提交后系统将自动生成救援任务。

提交后如何处理

提交救援申请后，系统会通知相关救援队成员，他们可以接受任务并开始救援行动。

救援行动

如何追踪救援行动

在救援行动页面中，用户可以看到所有正在进行的救援任务，并追踪每个任务的实时进展。

完成任务后如何反馈

救援行动完成后，队员可以通过系统提交救援反馈，详细描述救援过程和结果。

队员管理模块

队员注册

新队员可以通过此功能注册成为救援队成员。

队员信息

查看已注册队员的详细信息。

紧急通知模块

系统管理员可以通过该模块发布紧急通知，及时通知所有救援队成员突发情况。

系统设置模块

通过该模块，用户可以修改系统设置，包括个人信息、密码等。

常见问题解答
  • Q1：如何提交救援申请？

    在救援管理模块中选择“救援申请”，填写走失老人信息并提交。

  • Q2：如何注册成为救援队成员？

    在队员管理模块中选择“队员注册”，填写个人信息并提交，管理员审核通过后即可成为救援队成员。

  • Q3：如何查看当前救援任务？

    在救援管理模块中选择“救援行动”，查看当前救援任务详情。
